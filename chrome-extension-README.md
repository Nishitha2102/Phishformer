# Phishformer Chrome Extension

## Overview
Phishformer is a Chrome extension designed to help users identify phishing attempts and to enhance their online safety while navigating the web. This document provides a comprehensive overview of the extension's features, installation instructions, and usage guidelines.

## Features
- **Phishing Detection:** Automatically detects phishing links and provides alerts.
- **User Feedback:** Allows users to report suspected phishing attempts, contributing to the extension's learning database.
- **Alerts:** Notifies users when they are about to navigate to potentially harmful sites.
- **Dashboard:** Provides users with insights into their browsing habits and alerts history.

## Installation
1. Download or clone the repository: `git clone https://github.com/Nishitha2102/Phishformer`
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable "Developer mode" by toggling the switch in the top right corner.
4. Click on "Load unpacked" and select the directory where the Phishformer extension is located.
5. The extension should now be installed and active in your Chrome browser.

## Usage
- Once installed, the Phishformer icon will appear in the Chrome toolbar.
- Click on the icon to open the dashboard and view your alerts and browsing statistics.
- If you encounter a suspected phishing site, you can report it through the extension interface.
